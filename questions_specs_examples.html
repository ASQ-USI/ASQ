Introduction

File structure

Please use the following file structure and naming when creating presentatios for ASQ:

  • The presentation html-file must be in the root folder.
  • Include only one presentation. Do not include any other html-files.
  • Provide your css files within a /css folder. Name your main css file "style.css".
  • Provide your js files within a /js folder.

Please douple check that the zip file for upload does not include additional folders.

Question representation in ASQ

Questions or even better Assessment instances, can be inserted in a presentation through an HTML microformat or through the ASQ editor. HTML presentations that contains assessment instances in microformat are parsed and the questions are extracted and persisted.

Microformat

Format

Root Class Name

The root class name for assessment instances is assessment. This should be followed by a second classname that indicates the type of the assessment instance

Assessment instance types

  1. multi-choice
  2. text-input

Properties and Subproperties

The properties of an assessment instance are represented by elements inside the assessment instance. Elements with class names of the listed properties represent the values of those properties. Some properties have sub-properties, and those are represented by elements inside the elements for properties.

Property List

stem
The stem is the beginning part of the item that presents the item as a problem to be solved, a question asked of the respondent, or an incomplete statement to be completed, as well as any other relevant information
option
An element with the class attribute option defines a possible answer option for the assessment instance

multi-choice

multi-choice assessments specify the number of acceptable answers using the classes .choose<cardinality>. The available classes are:

Instance type Class Description
multi-choice .choose-1 only one of the provided options can be selected
.choose-0-n 0 to n options can be selected
.choose-1 1 to n options can be selected

text-input stem

The element with class .stem of a text-input assessment can have an attribute data-correct-answer='<value>' that specifies the correct answer.

option

Options can have a data-correct="{true,false}" attribute that specifies if the option is considered correct or wrong. The default value is false.
