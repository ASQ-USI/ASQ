We designed a tool to deliver slideshow right in your audience's browser

 Exciting! But that is not all...

What we really have is:

A portable solution for a lecturer to broadcast and collect feedback from the audience

Technologies used

More concretely

  • Node.js - Express server

  • Storage with MongoDB

  • Interface with Twitter Bootstrap

  • Impress.js to render the slideshow

  • Webscokets for syncronisation

Challenges

  • Designing (good) databases

  • Synchronization

  • Authentication and security

  • Archives and file system (User input)
